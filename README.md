# UIU Sanatani Vidyarthi Samsad (SVS) - Official Website

Welcome to the official repository for the **UIU Sanatani Vidyarthi Samsad (SVS)** community website. This project serves as a digital hub for the Sanatani community at United International University (UIU), showcasing culture, spirituality, and unity.

## üåü Project Overview

The UIU SVS website is designed to provide a vibrant experience of the Sanatani heritage. It features:
* **Event Tracking**: Real-time updates and countdowns for upcoming festivals like Saraswati Puja, Holi, and Shiv Ratri.
* **Media Gallery**: Integrated photo and video galleries documenting past celebrations and monthly prayers.
* **Membership Integration**: Embedded Google Forms for new members to join the legacy.
* **Dynamic Content**: Live feeds from Substack for blogs and YouTube for highlights.
